# 🚀 HireSense AI – Intelligent Resume Screening System

## 📌 Overview

HireSense AI is an NLP-powered system that automates resume screening by classifying roles, matching resumes with job descriptions, and ranking candidates using semantic similarity.

---

## 🔥 Features

* 📄 Resume Parsing (PDF/DOCX)
* 🧠 Role Classification (ML/NLP)
* 🔗 Resume–Job Matching (Cosine Similarity / BERT)
* 🏆 Candidate Ranking System
* 🧾 Skill Extraction
* 🖥️ Interactive Streamlit UI

---

## 🧠 Tech Stack

* Python
* Scikit-learn
* NLP (TF-IDF, BERT)
* Streamlit
* FastAPI (optional)

---

## 📊 How It Works

1. Upload resume
2. Extract & clean text
3. Convert to vectors
4. Match with job description
5. Rank candidates
